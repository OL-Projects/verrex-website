# Window Engineering Specifications — CAD Assembly Reference

## Core Rule
> **CHILD of animated group = MOVES with sash**
> **SIBLING of animated group = FIXED to frame**

---

## 1. Double-Hung (Vertical Slider) ✅
- **Moving**: Bottom sash (slides UP in channels)
- **On sash**: Lift handle (meeting rail), cam locks
- **On frame**: Balance channels, weatherstrip
- **Mechanism**: Spring-balanced vertical slide

## 2. Casement (Side-hinged, crank) ✅
- **Moving**: Left + right sashes (pivot outward on jamb hinges)
- **On sash**: Hinge leaves, lock keeper/strike
- **On frame**: Crank operator (at sill), hinge barrels, lock bodies
- **Mechanism**: Crank gear → scissor arm pushes sash outward

## 3. Tilt-Turn (European dual-mode)
- **Moving**: Single sash (tilts inward from bottom pivot)
- **On sash**: Espagnolette handle, hinge leaves
- **On frame**: Hinge barrels, strike plates
- **Mechanism**: Handle UP = tilt mode (bottom pivot + top restraint)

## 4. Awning (Top-hinged, pushes outward)
- **Moving**: Single sash (hinged at top, bottom swings out)
- **On sash**: Hinge leaves (sash frame + glass rotate together)
- **On frame**: Crank operator (at sill), hinge barrels
- **Mechanism**: Crank → scissor arm pushes sash bottom outward

## 5. Hopper (Bottom-hinged, tilts inward)
- **Moving**: Single sash (hinged at bottom, top tilts in)
- **On sash**: Lever handle (at top rail — you grab sash to pull in), hinge leaves
- **On frame**: Hinge barrels (at bottom sill)
- **Mechanism**: Pull handle → sash tilts inward

## 6. Skylight (Top-hinged, roof-mounted)
- **Moving**: Single sash (hinged at top, bottom pushes upward)
- **On sash**: Hinge leaves (sash frame + glass rotate together)
- **On frame**: Actuator/crank
- **Mechanism**: Mounted at ~30° roof angle, sash opens upward

## 7. Sliding (Horizontal Slider) ✅
- **Moving**: Right panel (slides left on track rails)
- **On sash**: Latch/lock handle
- **On frame**: Track rails, strike plate, fixed left panel

## 8. Jalousie (Louvre) ✅
- **Moving**: Glass slats (rotate on horizontal pins)
- **On slats**: (slats sit in metal clips)
- **On frame**: Crank operator (side jamb), linkage bar

## 9-15. Static Types (No mechanisms)
- Picture, Bay/Bow, Garden, Transom, Glass Block, Curtain Wall, Storefront
